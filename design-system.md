# Medical Clinic Website Design System

This document defines the **visual language, UI patterns, spacing, and component styling** that must be used consistently across the entire website.

The goal is to create a **premium medical clinic website** that feels expert, trustworthy, and caring.

# Testimonial Design

Use card or carousel layout.

Each testimonial includes:

Patient photo (optional)
Patient name
Star rating
Review text

Example review tone:

"Dr. Aanchal's diagnosis was spot on. I felt much better after her treatment and her care is exceptional."

# Form Design

Minimal
Clear labels

Name  
Phone  
Email

Message

Rounded borders  
Clear focus state  

---

# Footer Design

Footer must contain:

Clinic logo  
Quick links  
Services list  
Contact information  
Social icons

Footer background

Soft neutral tone

---

# Mobile Design Rules

Mobile layout must:

Stack sections vertically  
Use large touch-friendly buttons  
Avoid small text  

Hero title should shrink for mobile.

---

# UX Principles

Patients should be able to:

Understand services quickly  
Book appointment easily  
Trust the doctor immediately  

The **primary user goal** is:

Booking an appointment.

Every page should encourage this action.
